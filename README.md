# 🔢 NumPy Fundamentals

A complete, hands-on guide to NumPy — arrays, broadcasting, indexing, aggregations, statistics, matrix operations, and image data — with fully commented code and a study guide.

---

## 📂 Contents

| File | Description |
|---|---|
| [`numpy_fundamentals.py`](./numpy_fundamentals.py) | A single runnable Python script covering every topic below, with a comment explaining nearly every line. |
| [`NumPy_Complete_Guide.md`](./NumPy_Complete_Guide.md) | A structured, topic-by-topic study guide explained in plain language with examples. |
| [`NumPy_Complete_Guide.pdf`](./NumPy_Complete_Guide.pdf) | The same study guide as a PDF. |
| [`sample_image.png`](./sample_image.png) | Sample image used in the "reading images as arrays" section. |

---

## 📖 Topics Covered

- What NumPy is and why it matters for Data Science/ML
- Creating arrays (1D, 2D, 3D) and core properties: `shape`, `ndim`, `dtype`, `size`
- Three ways to create arrays: `zeros`, `ones`, `arange`, and random arrays
- Reproducibility with `np.random.seed()`
- Accessing elements (indexing)
- Matrix slicing across 2D, 3D, and 4D arrays
- Array manipulation — arithmetic operators and broadcasting
- Aggregations (`sum`, `max`, `min`) and Python vs. NumPy performance
- Statistics — mean, variance, and standard deviation
- Matrix multiplication — element-wise multiplication vs. the dot product
- Reshape and transpose
- A practical NumPy + Pandas example (calculating weekly sales totals)
- Comparison operators
- Sorting arrays — `sort`, `argsort`, `argmin`, `argmax`
- Reading images as NumPy arrays

## 🚀 How to Use

Clone the repo:
```bash
git clone https://github.com/<your-username>/numpy-fundamentals.git
cd numpy-fundamentals
```

Install dependencies:
```bash
pip install numpy pandas matplotlib
```

Run the script:
```bash
python3 numpy_fundamentals.py
```

Or open `NumPy_Complete_Guide.md` / `NumPy_Complete_Guide.pdf` for a topic-by-topic read-through with explanations.
